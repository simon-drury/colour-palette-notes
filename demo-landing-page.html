Slow Movement

The gradient moves very slowly (60 seconds per cycle) to create a subtle, sophisticated effect - not flashy or garish.

Technical Note: This design uses CSS gradients with blur filters to create a "backlight" effect. The borders, icons, and grids all share the same purple-to-green gradient animation, creating visual cohesion.

Beginner Tip: Look at the CSS comments above to understand how each part works! The key is the ::before pseudo-element with an animated gradient.
